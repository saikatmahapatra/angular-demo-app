Directives in Angular

An Attribute directive changes appearence and behaviour of a DOM element. There are 3 types of directive in angular

  • Component Directives: This is common directive
  • Structural Directives: change the DOM layout by adding and removing DOM elements. Example ngFor, ngIf
  • Attribute Directives: This is used as an attribute of HTML element. These are used to chnage appearence, behaviour of an component or another directive. Example - ngStyle
